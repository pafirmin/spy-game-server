## About

A back end for a web based version of the popular boardgame Codenames.

## Running the project

### Run the backend:

git clone https://github.com/pafirmin/spy-game-server.git

cd spy-game-server

npm run dev

### Run the front end:

git clone https://github.com/pafirmin/spy-game-client.git

cd spy-game-client

npm start
